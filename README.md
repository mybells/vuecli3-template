# vuecli3Template

基于 vue-cli3.1.3 的项目模板<br/>

技术栈主要使用 vue-cli3.0+vue+elementUI+vuex+vue-router+axios<br/>
并引入了 jquery,normalize.css,sass-loader

分支:<br/>
master: 只有前台模板（主线版本）<br/>
nodeclient: 包含前台和后台模板
